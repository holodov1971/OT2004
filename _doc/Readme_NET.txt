Работа с программой "Экзаменатор персонала" через сетевой диск.

В дальнейшем будет применяться терминология для идентификации компьютеров:

Второстепенный компьютер - экзаменатор (компьютер на котором сдают экзамены).
Главный компьютер - рабочий (компьютер на котором ведут базу данных).

1. На главном компьютере, на диске D, создадим и "расшарим" папку ТВ2004, правой кнопкой мыши на папке, выбираем вкладку "Доступ", на ней отмечаем пункты "Открыть общий доступ к этой папке" и "Разрешить изменение файлов по сети". На вкладке "Безопасность" дать полный доступ всем пользователям системы.

2.  На второстепенном компьютере на диске D создаём папку TB2004 в нее кладём три exe-файла:

EDITOR.EXE
Exam.exe
OT2004.exe 

А также в эту же папку кладём файлы шаблона Microsoft Word:

Лицензия.rtf
Протокол.dot
Телеграмма.dot
Удостоверения.dot

Тоже самое делаем и на главном компьютере.

На второстепенном компьютере должны обязательно присутствовать exe-файлы комплекса, т.к. запуск exe-файлов по сети через главный компьютер невозможен.
В папке ТВ2004 создаём папку data где должен лежать файл exam.ini в нём должна быть прописана строчка в конце:
ServerIB=имя_компьютера, где "имя компьютера" - полное имя главного компьютера в локальной сети.
Точно такую же строку прописываем в файле exam.ini в папке data на главном компьютере.

***Примечание: для операционной системы Windows 7 в свойствах всех exe-файлов, на вкладке "Совместимость" отметить пункты "Запустить программу в режиме совместимости с Windows XP (пакет обновления 3)" и "Выполнять эту программу от имени администратора".

3. На второстепенном компьютере создаём новый сетевой диск (Мой компьютер -> Сервис -> Подключить сетевой диск). Выбираем любую букву (например R), кроме тех что используются локальными дисками или сменными носителями, оставляем галочку на пункте "Восстанавливать при входе в систему". С помощью кнопки "Обзор" указываем путь к ранее расшареной папке на главном компьютере TB2004, жмём "ОК" и "Готово".	
Тоже самое делаем и на главном компьютере и указываем путь для сетевого диска: \\имя_компьютера\TB2004, где имя_компьютера - полное имя главного компьютера в локальной сети.

ВАЖНО!!! При назначении букв сетевым дискам на обеих компьютерах, назначать одну и ту же букву для двух компьютеров!!!

***Примечание: для операционной системы Windows 7 создавать сетевой диск не нужно.

4. На второстепенном компьютере устанавливаем и запускаем BDE Administrator ("Пуск" -> "Панель управления" -> "BDE Administrator") в списке алиасов находим алиас с названием "TB2004", нажимаем на него и в правом окне в строке "PATH" прописываем путь к базе данных на ранее созданном сетевом диске (R:\data), затем жмём Enter -> Ctrl+A -> ОК.
Тоже самое делаем и на главном компьютере, только путь прописываем на папку ТВ2004 хранящуюся на локальном диске (например D:\TB2004\data).

***Примечание: для операционной системы Windows 7 в BDE Administrator указываем путь к папке "data" как путь через сетевое подключение например: \\Имя_компьютера\TB2004\data

5. На главном и второстепенном компьютерах установить Firebird 2.5 как службу.

6. На главном компьютере в папке TB2004 создать новую папку "Pictures" куда будут помещаться рисунки для графических вопросов. Далее в будущем помещать все рисунки для графических вопросов в эту папку. На второстепенном компьютере при добавлении рисунков в графические вопросы, указывать путь на папку "Pictures", через сетевой диск (R:\Pictures\*.jpg). На главном компьютере тоже указывать пути к картинкам через сетевой диск, для корректного 
отображения картинок на обоих компьютерах.

7. На главном компьютере в папке TB2004 создать новую папку Word, куда будут автоматически помещаться файлы Microsoft Word.

***На месте при сдаче комплекса, найти и изменить пути существующих картинок для графических вопросов на путь через сетевой диск.

***ВАЖНО!!! Производить изменения в списках личного состава, формирование групп, списков подразделений и списков должностей, а также формирование билетов, только с рабочего компьютера, для сохранения изменений в базе данных.
Корректировка списка вопросов и настройку экзамена можно производить с обеих компьютеров, но желательно не одновременно!

***С второстепенного компьютера заходить на главный через рабочую группу "Pridn"!!!
